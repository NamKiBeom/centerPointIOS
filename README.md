# 멀티플랫폼을 위한 GIS 어플리케이션 기획 및 제작

> 기획

2017. 10 ~ 2017. 11

> 개발 기간

2017. 12 ~ 2018. 06

> Description

팀 프로젝트나 친구와 놀기 위해 약속 장소를 정할 때 만나는 장소가 누군가에겐 가깝고 누군가에겐 멀기 때문에 모두의 위치를 고려하여 중간지점을 찾아주는 어플리케이션을 기획하게 되었다.

> 사용한 API

Google Maps API

> 수상

제 5회 창의설계경진대회 우수상(교내)
